### Authentication with AWS Cognito Service
Add your AWS values to the `src/config` file to start
```
{
    "region": "YOUR_AWS_REGION",
    "userPoolId": "YOUR_COGNITO_USER_POOL_ID",
    "clientId": "YOUR_COGNITO_APP_CLIENT_ID"
}
```
